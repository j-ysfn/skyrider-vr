01 · Overview

SkyRider VR is a mobile VR flight simulation game built with A-Frame WebVR. You pilot a jet aircraft over a detailed city skyline — complete with a twin-tower landmark — entirely through head movement. Fly through glowing rings to score points and advance through two increasingly challenging levels.

02 · Installation & Setup

1
Play Online (Recommended)

Open the GitHub Pages link in your mobile browser:
https://yourusername.github.io/skyrider-vr/

2
Run Locally

Clone the repository and serve with any static file server. With Node.js installed: npx serve . — then open http://localhost:3000 in your browser.

3
Mobile VR (Google Cardboard)

Open the game on your phone browser. In the Setup screen, enable STEREO VR mode. Tap "Start Mission" and insert your phone into a Google Cardboard headset. The scene will split into stereo view automatically.

4
iOS Gyro Permission

On iOS 13+, you will be prompted to allow device motion access when the game starts. Tap Allow to enable head-tilt controls.

03 · Controls

The game is designed for gaze-based / head-tilt controls using your phone's gyroscope. No hands or controllers required.

08 · Technical Details

Engine: A-Frame 1.4.2 (WebVR/WebXR framework)
Rendering: Three.js (via A-Frame) with WebGL
Textures: Procedurally generated at runtime via HTML5 Canvas 2D API
Controls: DeviceOrientationEvent API (gyroscope), keyboard fallback
Audio: A-Frame sound component (Web Audio API)
VR: WebXR Device API — compatible with Google Cardboard, Meta Quest browser
